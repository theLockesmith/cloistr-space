# Cloistr Space

**Your Nostr-native productivity workspace.**

Activity. Projects. Social. All in one place.

## Quick Start

```bash
pnpm install
pnpm dev
```

Open http://localhost:5173

## Views

- **Activity** - Personal dashboard with files, tasks, calendar, mentions
- **Projects** - NIP-29 group workspaces with shared resources
- **Social** - WoT-filtered Nostr feed

## Features

- NIP-46/NIP-07 authentication
- NIP-0A synchronized contact lists (CRDT)
- Drive, Docs, Blossom integration
- WoT-filtered social feed
- Lightning zaps
